Projects in C++


FirstLeaf
[Started in July 2013, continuous development]
C++ engine for a 2D game using Allegro and Box2D.
It will kill three birds with one stone:
1. I'm learning C++, Box2D and Allegro in the deepest way possible allowed withouth being taught by industry professionals.
2. I'm learning how a whole C++ game engine works. I'm struggling, researching and solving real issues by myself.
3. It is serving as basis for the final project of my college degree. (See Zentract).
Link to repository (Requires my permission, feel free to ask me for it!)

Zentract
[Started in October 2013, expected finish in March 2014]
A branch of FirstLeaf for the final project of my college degree in the UPV.
The main topic of study is the usage and profit of a discrete event simulator instead of a main loop.
Working on it!
