Current projects

First Leaf [Started in June 2013]
C++ engine for a 2D game using Allegro and Box2D.
It will kill three birds with one stone:
It is serving as my final project for my college degree. (See Zentract).
It is making me learn C++ and Box2D in the deepest way I am able without being taught by industry professionals. Also Allegro.
It is forcing me to learn how a whole c++ game engine works, struggle with it and research and solve the problems.

Zentract [Started in October 2013]
A branch of First Leaf for the final project of my college degree.
The main difference is that it will be using a discrete event simulator instead of a main loop.
